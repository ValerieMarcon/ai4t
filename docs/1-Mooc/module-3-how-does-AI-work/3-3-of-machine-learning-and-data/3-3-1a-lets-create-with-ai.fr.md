# Activité : Créons avec l'IA
## Détection de l'IA : IA ou humain ?

L'activité suivante consiste à identifier si une image est réelle ou produite par une IA. Cette activité expérimentale est une introduction à la compréhension des GANs (Generative Adversarial Networks). Elle propose de discriminer les vrais portraits des portraits générés par le GAN (Source des images du GAN : https://thispersondoesnotexist.com Source des images des personnes réelles : https://pixabay.com/en/

Ce tutoriel peut être utilisé en classe.
